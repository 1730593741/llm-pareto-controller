Input JSON contains:
- state: Pareto-State metrics (convergence/diversity/feasibility signals)
- recent_experiences: recent state-action-reward transitions

1) Evaluate convergence, diversity, feasibility, and recent action outcomes.
2) Select EXACTLY one control_state from:
   - increase_diversity
   - increase_convergence
   - increase_feasibility
   - maintain_balance
3) Output concise rationale.

Return ONLY valid JSON with keys:
- control_state: string (one of the four values)
- reason: string
- convergence_signal: float (0~1)
- diversity_signal: float (0~1)
- feasibility_signal: float (0~1)
