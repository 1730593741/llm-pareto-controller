# LLM-Pareto-Controller

A research-oriented Python project for experimenting with closed-loop LLM control over NSGA-II.

## Quick start

```bash
python3.11 -m venv .venv
source .venv/bin/activate
pip install -U pip
pip install -r requirements.txt
python main.py
```

Expected output:

```text
project bootstrapped
```
