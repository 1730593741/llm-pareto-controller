# LLM-Pareto-Controller

Minimal M0 scaffold for a research-oriented Python project focused on LLM-guided multi-objective optimization.

## Quick start

```bash
python3.11 -m venv .venv
source .venv/bin/activate
pip install -U pip
pip install -r requirements.txt
python main.py
```

Expected output:

```text
project bootstrapped
```
